# FUTURE_PE_01 – AI Website Copy Generator for Local Businesses

## Task Overview
This project is part of the **Future Interns Prompt Engineering Track (PE)**.

The goal is to design a structured prompt system that generates high-quality, conversion-focused website copy for local businesses using AI tools.

---

## Objective
To solve a common problem faced by local businesses:

- Unclear website content
- Weak value proposition
- Poor call-to-action (CTA)
- Generic or outdated copy

This project uses **Prompt Engineering** to generate:

- Homepage content
- Services page descriptions
- Persuasive CTA sections
- Business-specific and engaging website copy

---

## Business Chosen
**GlowUp Beauty Salon – Chennai**

### Target Audience:
- Women (18–40)
- Brides
- Working professionals

### Services:
- Hair styling & cuts
- Skincare & facials
- Bridal makeup
- Spa & relaxation

---

## Tools Used
- ChatGPT (Prompt Design & Content Generation)
- Lovable AI (Website Generation)

---

## Prompt Engineering Approach

A structured prompt system was designed to:

- Take business inputs (name, location, services, audience)
- Generate organized website sections
- Adapt tone (friendly / professional)
- Focus on **benefits over features**
- Create **conversion-focused content**

### Prompt Levels:
1. **Base Prompt** – Website structure generation  
2. **Advanced Prompt** – Conversion optimization
